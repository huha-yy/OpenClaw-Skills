当前总体路线：

```text
第一阶段：热点监控到三平台内容发布包。
第二阶段：自动草稿、人工确认发布、低风险排期发布、评论收集与回复建议。
第三阶段：数据复盘、公司环境部署、多账号扩展。
```

当前推荐技术路线：

```text
OpenClaw 作为主编排中枢。
第一阶段不引入 n8n/Dify/Coze/LangGraph。
第二阶段视复杂度评估 LangGraph 和 n8n。
图片策略采用图文协同方案优先，Pexels/图库检索次之，ComfyUI/Stable Diffusion 作为生成式补充。
```

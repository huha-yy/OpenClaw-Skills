# 阶段 3：发布包与审核计划

## 1. 阶段目标

本阶段目标是让运营真正看得懂、用得上系统输出。

第一版发布包不是简单文案，而是“待审核内容交付包”。

## 2. 发布包定义

发布包包含：

```text
1. 热点来源
2. 热点摘要
3. 相关性判断
4. 内容策略
5. 小红书图文内容包
6. 抖音视频内容包
7. 微信公众号图文内容包
8. 风险报告
9. 发布元数据
```

## 3. 小红书图文内容包

## 6. 审核规则

第一版审核分为三类：

### 6.1 事实审核

检查：

```text
来源是否可靠
事实是否有多个来源支持
是否存在未经证实的信息
是否有时间、地点、政策范围错误
```

### 6.2 品牌审核

检查：

```text
是否符合公司品牌语气
是否过度营销
是否攻击竞品
是否包含公司禁用表达
```

### 6.3 平台与合规审核

检查：

```text
是否涉及广告法风险词
是否涉及敏感行业表达
是否适合对应平台风格
是否需要人工二次确认
```

## 7. 风险分级

```text
low：低风险，可人工快速确认
medium：中风险，需要运营仔细审核
high：高风险，不建议自动生成发布内容，只做提醒
```
